## 使用技術

- HTML5
- CSS3
- JavaScript（Vanilla JS）
- GitHub Pages（公開）

---

## 実装機能

### レスポンシブナビゲーション
- スマートフォン表示でハンバーガーメニューを実装
- JavaScriptで開閉制御

### スクロールアニメーション
- IntersectionObserver を使用
- 要素が画面に入った時にアニメーション表示

### ダークモード切替機能
- ボタンでテーマ切替
- localStorage を使用して状態を保存

### 現在ページのナビ表示
- JavaScriptで現在ページを判定
- 該当ナビに下線を表示

### お問い合わせフォームUI
- 実務を想定したフォームデザイン
- フォーカス時の視認性向上
